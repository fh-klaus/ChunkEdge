# FAQ

Frequently asked questions about ChunkEdge.

## Can I use ChunkEdge as a replacement for a Vanilla Minecraft Server?

No.

ChunkEdge is a Minecraft server _framework_. The goal of ChunkEdge is to provide a platform for developers to build custom Minecraft servers. **ChunkEdge, on its own, is not a direct replacement for a Vanilla Minecraft server.** However, it should be flexible enough to build a Vanilla Minecraft server on top of it. You just have to implement the game logic yourself.

ChunkEdge is best suited for building custom experience servers such as minigames and role-playing servers with content completely different from Vanilla Minecraft.
